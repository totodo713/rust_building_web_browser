### Set up

1. `git` インストール
    ```bash
    sudo apt install -y git-all
    ```
2. `QEMU` インストール
    ```bash
    sudo apt install -y qemu-system
    ```
3. `Rust` インストール
    ```bash
    mise install
    ```

### 操作

* 起動

    ```bash
     ./run_on_wasabi.sh
    ```
* 停止
    * `Ctrl+c`で終了させる。
